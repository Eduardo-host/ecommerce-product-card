# 🛒 Ecommerce Product Card

A clean and modern Nike product card UI built with **HTML**, **CSS**, and **JavaScript**.

This mini project simulates a simple ecommerce product card with quantity controls, cart interaction, rating stars, and a clear cart feature. It was built as a practice project to improve UI design, DOM manipulation, and basic JavaScript logic.

## 🚀 Features

- Modern product card layout
- Nike logo and product image
- Product title and description
- Star rating section
- Product price
- Quantity selector with plus and minus buttons
- Add to cart interaction
- Cart badge counter
- Clear cart / trash button
- Smooth hover effects
- Clean and responsive design

⚙️ How It Works

The user can increase or decrease the product quantity using the + and - buttons.

When the user clicks Add to cart, the selected quantity is added to the cart badge.

The trash button clears the cart and resets the selected quantity back to 1.
